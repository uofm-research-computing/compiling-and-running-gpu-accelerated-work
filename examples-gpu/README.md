# Samples
Some simple samples to understand the GPU compilation options. This is targeted to BigBlue, but can be modified to work on itiger or other GPU clusters.

## cmake
Modifying an existing cmake build, nbody, to compile on bigblue.

## mpi  
Using MPI and OpenMP target offloading to use GPUs. This one pulls from another group's tutorials.

## nvcc  
Simple nvcc compilation.

## openmp
Simple nvc++ compilation.
